# User Directory Dashboard

## 🚀 Live Features

* Display users in responsive card layout
* Search users by name or email
* Sort users by name and company (ASC/DESC)
* View detailed user information
* Modern UI with Bootstrap, icons, and avatars

## 🛠 Tech Stack

* React.js
* Bootstrap
* React Router
* JSONPlaceholder API

## 📦 Installation

```bash
npm install
npm run dev
```

## 🌐 API Used

https://jsonplaceholder.typicode.com/users

## 📸 Demo

Click on a user card to view full details.
